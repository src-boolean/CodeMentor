# Курсовая работа: CodeMentor
**Студент:** Иванов Арсений, группа 411ИС-22
**Тема:** Приложение для изучения программирования с использованием AI

## О приложении
CodeMentor — это тренажер по C++ и Assembler. Приложение не просто показывает готовые задачи, а генерирует их через нейросеть Google Gemini. Можно выбрать тему, решить задачу прямо в редакторе на телефоне и получить проверку кода с объяснением ошибок.

## Что реализовано
1. **Регистрация и вход.** Создание аккаунта, данные сохраняются в базу.
2. **Генерация задач.** Выбираешь язык и сложность -> получаешь условие.
3. **Проверка решений.** Отправка кода нейросети, ответ приходит с разбором ошибок.
4. **Чат-помощник.** Если сложно, можно спросить подсказку у бота.
5. **Локальная БД.** История решений и профиль работают через SQLite, данные не теряются при закрытии.
6. **Система уровней.** За решения дается опыт, повышается ранг.

## Используемые технологии
*   **Язык:** Kotlin
*   **Среда:** Android Studio
*   **Сеть:** Библиотека Retrofit 2 (для запросов к API Gemini).
*   **База данных:** SQLite.
*   **Дизайн:** Стандартные элементы Material Design.

## Как запустить
1. Установите APK файл на телефон (Android 8+).
2. **Обязательно включите VPN.** Так как приложение делает запросы к серверам Google, без VPN функции генерации и проверки задач работать не будут (будет ошибка подключения).
3. Вход и просмотр истории работают без интернета.
